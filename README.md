# Bus Reservation System (Java + JDBC)

A console-based Bus Reservation System developed using **Java** and **JDBC**, implementing database interaction with MySQL. This project demonstrates core Java concepts, object-oriented programming, and real-world database connectivity.

---

## Features

* View available buses
* Book bus tickets
* Check seat availability
* Store bookings in MySQL database
* Menu-driven console interface
* Clean DAO-based architecture

---

## Tech Stack

* Java (Core Java)
* JDBC
* MySQL
* OOP Principles

---

## Project Structure

```
Bus_Reservation/
│
├── MainClass.java        # Application entry point (menu/UI controller)
├── DB_Connection.java    # Database connection handler
├── Bus.java              # Bus model class
├── Booking.java          # Booking model class
├── BusDAO.java           # Bus database operations
├── BookingDAO.java       # Booking database operations
```
